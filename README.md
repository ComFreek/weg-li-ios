## Setup

* Install latest Xcode version via macOS App Store

### Fastlane

* Install [Homebrew](https://brew.sh/)
* Install [Ruby](https://www.ruby-lang.org/) via Homebrew

```shell
brew install ruby
```

* Install [Bundler](https://bundler.io) via Ruby

```shell
gem install bundler
```

* Install Ruby dependencies

```shell
bundle install
```
